![](img/f33d2f7699b941e64719366f839e655a.png)

Chakra UI 是一个组件库，为您提供创建 React 应用程序所需的构建模块。它简单、模块化、易于使用。它提供了像黑暗模式、样式简易、可访问性等特性。它在 Github 上有超过 26k 颗星。
