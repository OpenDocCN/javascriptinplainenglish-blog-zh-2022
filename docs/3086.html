具有高级定制功能的强大虚拟数据网格智能表。excel的最佳功能加上令人难以置信的性能。

RevoGrid组件有助于以数据表“excel like”或列表的形式表示大量数据。最重要的是，它提供了内置的范围编辑或每个单元格的编辑，键盘支持和自定义编辑和渲染功能。可以在任何主要框架中工作，也可以不使用任何框架。

这个有更多可用的框架环境，包括Vue，Svelte，React，Angular，Ember。

特点:

8。jQWidgets。

文档

jQWidgets Data Grid for React是一个用React、Typescript和jQWidgets框架构建的专业数据网格组件。它提供了数百种功能，如排序、过滤、分组、主从视图、分页等。它还附带了许多高级功能，如拖放、列大小调整、打印、数据导出、聚合、RTL支持等。React数据网格是用React UI框架构建的企业应用程序的完美解决方案。

它还可以导出和打印所有可编辑的功能。
