要在 JS 中创建一个对象，您需要遵守一些规则:

*   如果你的类有属性，一个`***constructor***` 是必须的。否则 JS 会自动创建一个空的，没必要显式添加。
*   Classe 构造函数有确切的名字`***constructor***`。
*   要初始化该类，请使用`**constructor**`。
*   要定义类方法，您需要将它们添加到类中。类方法只是普通的 JS 方法。
