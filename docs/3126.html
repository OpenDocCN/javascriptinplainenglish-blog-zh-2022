为网络开发者提供10多种独特的Chrome扩展

原文：https://javascript.plainenglish.io/to-build-web-applications-smarter-easier-better-and-faster-you-will-need-some-browser-1a40f911c877?source=collection_archive---------12-----------------------#2022-08-02

使用这些Chrome扩展为前端开发人员更快、更好、更容易和更智能地构建web应用程序。

为了更智能、更容易、更好、更快地构建web应用程序，您需要一些浏览器扩展，尤其是如果您是前端开发人员的话。好吧，你很幸运，因为在这篇文章中，我将向你展示“10+为Web开发者设计的独特的Chrome扩展”。

1. JSONViewer

在浏览器上查看JSON可能对我们的眼睛非常有害，但是这个扩展突出显示了您在浏览器中打开的任何Json文件，从而使它更容易阅读。

2. Colorzilla

这是最受欢迎的扩展之一，拥有超过2M+用户。这个扩展主要用于从页面中提取颜色。

3.氨基

Amino是一个免费的浏览器扩展，可以让你在任何网页上添加自定义CSS。有了Amino，你可以定制颜色、字体、页面布局等等。这是相当先进的，它也有一个CSS检查工具。

4.子块原点

uBlock Origin是一款广告拦截器，可以完美有效地拦截广告。我强烈推荐这款广告拦截器；因为它是业内最好的之一。它有超过1000万的下载量，所以如果你讨厌广告，试试这个。

5. Wappalyzer

Wappalyzer是一个技术剖析器，它向你展示网站是用什么构建的。它可以让你找到框架、电子商务平台、JavaScript库等等

6.什么字体

你有没有从网页上偶然发现过很多漂亮的字体？WhatFont是一个可以让你从网页中识别字体的工具。

7.窗口大小调整器

这一点对于实现响应式设计非常重要。窗口Resizer是一个工具，让您模拟各种屏幕分辨率。

8.网络开发者

Web developer是一个扩展，它为编写和测试网站提供了各种重要的工具。

9.反应/真空/角度开发工具

这些扩展为您使用React、Vue.js和Angular等前端JavaScript框架提供了很大的帮助。

10.每日开发

daily.dev是一个很棒的在线社区，开发者可以通过它了解最新的开发者新闻。有了这个扩展，当你打开一个新的chrome标签时，daily.dev将自动成为你的默认主页。

11. CSS查看器

CSS Viewer允许你点击网页上的任何按钮/图片/文本等，你可以立即看到所使用的CSS代码。
