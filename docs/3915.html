第2部分:设计一个打开面板的按钮

在这一部分，我们将在文本旁边的弹出按钮上添加一个向右的v形图标。

当弹出面板打开时，图标将指向底部，否则将指向右侧。

首先，我们必须安装react heroicon。

npm install @heroicons/react

然后我们导入顶部的图标。

import { ChevronRightIcon } from '@heroicons/react/solid'

{({open})=>(<></>)}
