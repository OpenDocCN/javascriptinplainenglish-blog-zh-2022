让我们先来解释一下switch语句是如何工作的。(来源:MDN):

一个 switch 语句首先对其表达式求值。 然后查找第一个 case 子句，其表达式的计算结果与输入表达式的结果相同(使用 严格比较 、===)并将控制转移到该子句，执行相关语句。(如果多个 case 与提供的值匹配，则选择第一个匹配的 case ，即使 case 彼此不相等。)

这到底意味着什么？换句话说，当使用switch(true)模式时，它会在评估每个case子句后将其与true进行比较。这是这种模式中额外的、无意义但不可避免的比较。
