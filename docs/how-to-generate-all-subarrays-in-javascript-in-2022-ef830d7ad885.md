## 问题陈述

给定一个数组，用迭代的方式生成给定数组的所有可能的子数组。

```
Input : [1, 2, 3]

Output: [1], [1, 2], [1, 2, 3], [2], [2, 3], [3]

Input: [1, 2]

Output: [1], [1, 2], [2]
```
