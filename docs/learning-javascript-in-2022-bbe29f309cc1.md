# 面向初学者的 JavaScript 概念

1.  JavaScript 的单线程执行模型
2.  JavaScript 模式:模块和原型模式
3.  JavaScript 闭包和范围
4.  JavaScript 中的链接
5.  原型链接的概念
6.  功能和可变提升
7.  JavaScript 中的不变性
8.  高阶函数
9.  函数式编程及其优势
10.  生活和单身

上面给出的是为了有效地开始 JavaScript 开发应该关注的核心概念和主题。有很多概念需要学习，但是在进一步学习任何前端或后端语言或框架之前，您应该确保您已经深入了解了这些主题。
