# 7.直流 js

![](img/23399c0094d8f659ef52851a42383ac6.png)

这是一个多维图表，旨在使用 d3.js 呈现的 cross filter 进行本地工作。在这个库中，每个图表通过其元素的位置、大小和颜色显示一些属性的集合，还提供了一个可以过滤的维度。当滤镜或笔刷更改时，所有其他图表都会使用动画转场进行动态更新。这个库在 GitHub 上有超过 7k 的星星。
