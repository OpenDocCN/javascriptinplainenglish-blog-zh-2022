现代 web 开发是基于组件的。想出一个引人注目的设计可能很有挑战性，尤其是如果你是一个经验不足的 web 开发人员。

在本文中，我编译了一些我最喜欢的组件，您可以用它们来启发自己的项目。为了提供多样性，我包含了从按钮、栏和菜单到卡片、日历和神经形态元素的项目。

对于每个组件，我将包括代码的直接链接，嵌入式交互式预览，以及作者链接，这样您也可以检查他们的其他工作。
