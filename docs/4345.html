用单行代码将2d数组转换为1d数组

这篇短文将用一行代码解释如何将2D数组转换成1D数组。基本上，你需要为2D数组使用 Array.prototype.concat() 和 spread运算符。

用2D数列检查下面的例子:

const numbers = [[1,2], [2,3], [3,4]];
const result = [].concat(...numbers)

// result: [ 1, 2, 2, 3, 3, 4 ]
