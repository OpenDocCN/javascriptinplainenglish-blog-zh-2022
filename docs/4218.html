我们如何在不破坏用户体验的情况下将代码插入媒体编辑器。

这篇文章，我是在做了大量的研究和尝试了许多方法来给中型帖子添加代码后写的。

方法1:使用内置代码块。

将代码添加到中型post的简单方法是通过selecting the text + backtick ( ` ) for inline code和

3 backticks (```) for block code.

挑战:

  • 格式化代码非常困难。
  • 代码是用灰色写的，很难区分关键字、变量和表达式等。
