# 先决条件

*   [*节点 JS 安装在*机器上](https://nodejs.org/en/)。

先决条件列表中没有太多内容:)

# 让我们像开发者一样用谷歌搜索

*   步骤 1:安装 npm 软件包

```
$ npm i -g glook
```

*   第二步:所有的事情都已经准备好了

```
$ glook SEARCH_TERM
```

现在，你可以从终端直接谷歌搜索。
