前端开发

前端开发是一种风格的计算机编程，专注于编码和创建用户可以看到的网站元素和功能。它是关于确保一个网站的视觉方面的功能

后端

后端是网站的服务器端。它存储和安排数据，并确保网站客户端的一切正常运行。这是网站的一部分，你看不到也无法与之互动。它是软件中不与用户直接接触的部分

  3. Ruby on Rails
  4. Java (Spring)
