#过滤器

filter方法用满足条件的元素创建一个新数组。

注意:filter方法不会修改原始数组。

#地图

map()方法创建一个新的数组，其中填充了调用数组中每个元素的函数的结果。

# ForEach

forEach()方法为每个数组元素执行一次提供的函数。

#包括

如果选择的值包含在数组中，includes() 方法返回true或false。

#寻找

find()方法返回所提供数组中满足所提供条件的第一个元素。

# FindIndex

findIndex()方法返回所提供数组中满足所提供条件的第一个元素的索引。

#减少

reduce方法用于将数组缩减为单个值，并从左到右为数组的每个值执行一个提供的值。

对于每一步(从左到右)，函数的返回值都存储在累加器变量中。

#一些

如果数组中至少有一个元素满足给定的条件，some()方法返回true。

#每

如果数组中的所有元素都满足给定的条件，则every()方法返回true。
