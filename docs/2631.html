常量断言

在Typescript 3.4中，Typescript为文字值引入了一个新的构造，称为 const assertions 。对我来说，这个特性很长一段时间都没有被发现。但是最近，它对我来说非常方便。这是提供类型安全的类似枚举的模式的一种很好的方式。

它的语法是用const代替类型名的类型断言(例如let x = 123 as const)。当使用const断言时，我们告诉Typescript去推断它能为一个表达式指定的最具体的类型。如果您关闭它，编译器将使用其默认的类型推断行为，可能会产生一个更宽的或更通用的类型(例如，字符串或数字)。这听起来可能很复杂，但是const断言很简单，最好用一个例子来解释。让我们看看这个简单的变量:

Typescript利用其自动类型检测。myName变量的类型被推断为最通用的类型string。
