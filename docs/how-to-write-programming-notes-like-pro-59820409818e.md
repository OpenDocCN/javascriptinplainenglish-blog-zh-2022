我仍然记得我开始编程之旅的那些日子。老实说，这太可怕了，我完全乱了套。但是，我从来没有放弃自己，继续编程，直到我获得专业知识。

在学习一门新的编程语言时，许多人都面临着许多障碍。主要我们可以把它分成两部分。一个是资源，第二个是记录你正在学习的东西。

毫无疑问，我会给你一些写笔记的提示和技巧，这对你会很有用。但在开始这个话题之前，我建议你使用在线工具，如谷歌 Jam board 或谷歌文档，这样你就可以随时随地访问你的笔记。在某种程度上节约纸张也是有帮助的。

## K.国际安全系统

这对所有初学者和专业人士来说都是最重要的提示。“保持简单和愚蠢”是写下一些精彩笔记的最好方法，你也可以快速复习这些笔记。

它主要包括三个步骤:

*   理论
*   图表
*   短伪码

首先，写下你正在研究的主题的一个小理论，然后画出它的图表。是的，你必须在笔记中包括一个图表。如果你正在学习数据结构，那么你必须在必要的地方加入图表。

在这之后，写下你正在学习的概念的一小段伪代码或功能。

到目前为止，这是我在学习编程时用来写笔记的最好的方法。下面是我的意思的简短说明:

```
Array-
Array are the contagious memory locations that store same kind of data types. *DIAGRAM OF ARRAY* |0|1|2|3|Code int a[4];
it is way to declare array which stores int data
```

除了这几个非常有用的快速提示之外，还有:

*   不要写太多，内容少一点好把握
*   使用不同颜色的笔
*   使用规则 K.I.S.S
*   把你的笔记放在手边
*   每周至少修改一次笔记，以获得这个概念的专业知识
*   不要压力太大
*   中间休息一下
*   使用在线资源和流行的 YouTube 频道学习编程，而不是付费课程
*   更频繁地在编译器中键入代码
*   一旦获得专业知识，就可以自己实现数据结构代码
*   不断学习更新更新的概念，不断成长
