## 为您的设计灵感精心挑选的 UI 组件集。

![](img/1e879ae5fd2003402997add3596bf633.png)

现代 web 开发已经转向基于组件的体系结构，这导致了更快的开发、更多的控制和更低的维护成本。在这篇文章中，我精选了一些我最喜欢的 UI 组件来激发你的设计灵感。

我试图在我们的开发工作流程中包含不同类型的一些最常用的组件，包括卡片、传送带、按钮、复选框、图标、加载器和菜单。

对于每个组件，我将提供一个直接链接，一个交互式预览，这样您就可以在运行中获得初步印象以及作者链接，这样您也可以探索他们的更多工作。
