TXT记录的正则表达式是什么？

原文：https://javascript.plainenglish.io/whats-the-regular-expression-of-txt-record-897d54841a43?source=collection_archive---------20-----------------------#2022-01-17

TXT记录可以在下面输入关键字。

a-z  
A-Z  
0-9  
Space  
- (hyphen)  
! " # $ % & ' ( ) * + , - / : ; < = > ? @ [ \ ] ^ _ ` { | } ~ .

允许你只输入这个的正则表达式是什么？

底部是我做的。

————————回答——————

const txt_rex = /[#\&\\+\-%@=\/\\\:;,\.\'\"\^`~\_|\!\/\?\*$#<>()\[\]\{\}]/i;const txt_rex = /^[\w !"#$%&'()*+,\/:;<=>?@[\]^`{|}~.-]+$/;

\w = 'A-Za-z0-9_'
[...]+ =一个字符集，1个或多个
^...$ =从开始...直到结束

HTML示例:

input:invalid{box-shadow: 0 0 0 4px red;}<input pattern="[\w !\x22#$%&'()*+,/:;<=>?@[\]^`{|}~.-]+">
