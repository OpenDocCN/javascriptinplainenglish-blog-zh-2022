模板清理

如果我们有很多实用程序，想把它们移出模板，该怎么办？通过 Stylify 你可以使用可重复使用的组件来实现。它们可以在使用它们的文件或全局配置中的注释内定义(需要不带括号的js对象)。

在生产中，组件也会被损坏。

语法解释

在上面的例子中，你可以看到使用类CSS选择器的Stylify。有一些不同。

  • _选择器是用来代替空格的
  • [span]{display:inline-flex}是一个内嵌的自定义选择器。这允许您定制选择器的样式。
  • &内部[&:hover_span:first-of-type]总是指像在SCSS那样的上层
  • 组件中的缩进语法也类似于SCSS。为了简单起见，它只支持嵌套和链接
span { 
  display:inline-flex 
}
&:hover span:first-of-type {  
  transform:scale(1.5) 
}
