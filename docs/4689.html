在您的谷歌搜索结果中获得StackOverflow答案

假设你是一名开发人员，想谷歌一些东西。

我猜你的过程看起来像这样:

  1. 在谷歌中写一个查询
  2. 向下滚动到第一个StackOverflow命中
  3. 点击链接，等待页面，忽略实际问题，向下滚动到答案
  4. 阅读答案

我做了一个叫做 StackPop 的工具，它会让你成为谷歌之神。从现在开始你只需要两步:

  1. 在谷歌中写一个查询
  2. 阅读答案

缺点？

我能想到的唯一负面影响是，如果你不喜欢来自StackOverflow的答案，它会在你的搜索列表中占据一些空间。

如果你看到任何缺点，请报告。我没有注意到任何速度或其他方面的问题。您通常的搜索结果将仍然存在，不会删除任何内容。
