# 什么是接口？

接口是与**面向对象编程**相关的概念。

> 接口是没有实现代码的类。它只有定义类所必需的代码。

这就像一个**契约**，我们创建它来声明**如果你想创建这个类**需要实现什么。但是没有说应该如何实现，因为我们没有实现代码。
