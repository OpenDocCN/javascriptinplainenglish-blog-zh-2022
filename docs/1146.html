将混淆器作为模块/中间件包安装

你可以简单地用Yarn或NPM安装这个包，并把它添加到你的依赖项中。在https://obfuscator.io/随意检查全部命令

$ npm install --save-dev javascript-obfuscator
or
$ yarn add --dev javascript-obfuscator
