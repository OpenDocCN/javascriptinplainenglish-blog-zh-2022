要成为一名开发人员，你不需要参加任何昂贵的训练营，或者从任何大学获得昂贵的计算机科学学位。

如果你想学习编码。您可以使用免费资源，成为一名优秀的开发人员。

YouTube将成为你的免费大学。

Reddit和Twitter将是你需要交友和支持的社区。栈溢出将网站清除您所有的疑问和疑虑。

够了。在这里我给你做了一个逐月学习编程的计划。

第一个月

  1. 选择任何编程语言。不要浪费时间去过多考虑应该使用哪种编程语言。所有编程语言的基础都是一样的。
  2. 如果启动Python，Java或者C++会更好。
  3. Java学习资源— Java 8初学者全教程
  4. 学习Python的资源— Python初学者全教程
  5. 你这个月的目标是关注任何编程语言的基础。你不应该一次专注于一种以上的编程语言。如你所想，你会混淆语法。
  6. 不要光看教程就以为自己已经完成了课程。你和导师一起编码。你必须尽自己的努力。您需要进行一些代码更改并查看结果。

第二和第三个月

  1. 学习数据结构和算法。
  2. 可以通过网络课程。你甚至可以在书籍的帮助下做到。
  3. 你正在学习这一点，因为你将在各种公司面试中被测试数据结构和算法技能。
  4. 一个很好的资源是麻省理工学院的讲座。这是一个很好的资源，可以帮助你理清基础知识。教过这些课程的教授对这门学科了如指掌。
  5. 你可以在这里找到课程— 算法和计算
  6. 你不应该一次看完整个课程。你只需要看一场讲座，然后去Hackerrank 上解决相关的问题。你也可以选择leet code上的简单问题。
  7. 记住，你不可能一下子解决所有的问题。习惯这些问题需要一些时间。
  8. 有些情况下，你甚至不能正确理解这个问题。在这些情况下，你不应该失去耐心。有一些问题是和各种话题混搭的。
  9. 这两个月将确保你获得解决问题的技能。
  10. 确保你不会失去希望。如果你是初学者，这个阶段对你来说会有点困难。

第4个月

  1. 一旦你完成了麻省理工学院讲座中所有的算法和数据结构视频。
  2. 你对计算机算法的基础有很深的理解。在这一个月里，如果需要的话，可以快速重温所有的讲座。你必须这么做。
  3. 现在剩下的时间应该用来练习你在课程中学到的所有概念的问题。
  4. 有成千上万的网站供你练习做题。你只要选一两个，不断练习题就行了。
  5. 全球使用的两大网站包括— Codeforces 和 Leetcode
  6. 集中精力解决更多的问题，用你学到的所有概念解决更多的问题。

第5个月

  1. 即使是这一个月，你也要花一半的时间多练习数据结构和算法方面的问题。
  2. 你可能想知道为什么我如此关注数据结构和算法。因为这是唯一会在几家公司进行测试的东西。
  3. 在任何大型科技公司，这都是成为软件开发人员最难也是最重要的部分。这里还有两个练习问题的网站——hacker rank、 Codechef
  4. 在这一个月里，你需要弄清楚自己想成为什么样的开发者。成为软件开发人员有很多途径。
  5. 你可以选择前端开发，后端开发，iOS开发，或者如果你想用云。这个月致力于发现你的兴趣，选择你感兴趣的东西。
  6. 即使这个月你下不了决心。别担心，我们将在第七个月讨论成为前端、后端和iOS开发者的途径。

第6个月

  1. 你应该花这个月的时间学习如何制作网站的基础知识。
  2. 对软件开发者来说，了解网络是如何工作的是很重要的。你应该从基本的HTML、CSS和JavaScript开始。不要跳过这一步，即使你不想成为全职前端工程师。
  3. 作为一名软件开发人员，其他开发人员希望您至少了解HTML、CSS和JavaScript的基础知识。
  4. 在Youtube上，有成千上万的教程，你可以从中学习这些东西。你可以选择任何资源。当我第一次知道youtube上没有多少视频时，我参考了这个标准。
  5. 这个月创建基本的网站。如果英语不是你熟悉的语言，你可以选择母语的视频。因为世界各地的导师都可以在youtube上找到。
  • 学HTML— HTML全教程建网站教程
  • 学习HTML5和CSS3 — 从头开始学习HTML5和CSS3
  • 学习JavaScript — JavaScript编程全程

第7个月

  1. 这个月要学的第一件事是关于版本控制。您应该了解版本控制是如何工作的。
  2. Git和Github是前进的方向。先从版本控制的基础开始，再详细了解GitHub。理解Github很重要。
  3. 有关Github的更多信息，请参见—Git和GitHub for Poets简介。这是一个关于Git和Github的小老视频。但是老师在帮助你学习概念方面做得很好。
  4. 这里还有一个来自Freecodecamp学习GitHub的视频——Git和GitHub初学者速成班
  5. 一旦你学会了如何使用版本控制，你就会感到自信。学习Git和Github的时间不会超过10天。
  6. 这个月我们有很多空闲时间。记住，我们已经讨论过决定你想成为什么样的开发人员。

前端开发人员

  • 因为你已经在上面详细介绍了HTML、CSS和JavaScript的基础知识。你在旅程中遥遥领先。
  • 这一次你需要回顾你之前学过的所有JavaScript概念。现在你需要详细地覆盖它。
  • 你需要了解承诺、闭包、生成器以及任何函数的“this”范围。
  • 当你学完JavaScript，你应该学习React，Angular或Vue.js之间的前端框架
  • 你可以选择这三个中的任何一个，但是最近对React开发者的需求似乎更多。此外，这是我的最爱之一，是的，你可以说我偏爱React。

  3. 初学者Vue.js课程
  4. 学习Angular —全教程教程
  5. React JS — React初学者教程
  • 你应该学习任何框架，并尝试建立可以向各个公司的招聘人员展示的项目。
  • 好的项目像魔法一样有效。
  • 这里有一个很好的Github库，它包含了在启动你的网站之前你需要拥有或测试的所有项目的列表— 前端清单

  4. 有些项目是绝对不能跳过的。该存储库也涵盖了这些项目。

后端开发人员

  • 第六个月，你已经学会了HTML，CSS，JavaScript的基础知识。这将帮助您踏上成为后端开发人员的旅程。
  • 你应该从复习之前学过的所有概念开始这个月。一旦你完成了审查。你可以在Django和NodeJs之间选择。
  • 为什么我要说在这两者中选一个？因为对于初学者来说，它们有点容易学。如果你选择Django，你必须先学习Python，然后只有你才能晋级。
  • 如果选择Node.js，需要详细学习JavaScript。这几天对Node.js开发者的需求频繁增加。因为许多公司都想雇佣一名JavaScript开发人员。

iOS开发

  • 学习iOS开发，首先需要学习swift。Swift是一种为编写iOS应用程序而开发的编程语言。
  • 我不是iOS开发方面的专家。我还没有用swift编程语言写过一行代码。
  • 如果你想为iOS设备开发应用程序，你首先需要找到合适的人来指导你完成整个旅程。

第八月

  • 这个月致力于建设项目。
  • 你需要用你到目前为止学到的知识来构建项目，把每一行代码放到Github 上。
  • 一旦你所有的代码都可以在GitHub上获得，你就可以和任何人分享。
  • 您还可以使用其他工具来托管一些实时项目。在网上直播这个项目时，你会学到很多新东西。
  • 你也可以为开源项目做贡献。有几个项目已经被公司开源，他们想改善软件。
  • 你可以开始为开源做贡献，并建立一个很好的帮助你的人的网络。
  • 无论你在做什么，都要公开分享。使用 Twitter 、 LinkedIn 或者在各种平台上写博文。
  • 公开分享会吸引招聘人员。各个公司的招聘人员活跃在这些平台上，寻找有素质的人。
  • 写博客和写下你不断学习的东西肯定会对你有利。
  • 这里有一个优秀的 Github库来帮助你一个应用的真实例子。— 现实世界

第九个月和第十个月

  1. 如果你已经开始为开源做贡献，继续你的旅程，公开分享你的工作。学习将很快复合，人们将开始向你寻求帮助。
  2. 此外，你必须了解计算机科学的基础知识。这在你成为开发人员的过程中很重要。
  3. 这里我提到了所有你应该学习的重要话题。如果您想详细探讨任何主题，请随意。

面向对象编程概念:OOP概念。

面向对象编程(C++)中涉及的重要主题

  1. 哎呀的概念。
  2. 多态性的类型。
  3. 虚函数——运行时多态性。
  4. 继承——类型、虚类、可怕的钻石问题(Imp)。
  5. 构造函数和析构函数——虚拟析构函数、私有构造函数和析构函数。
  6. 智能指针。
  7. 单例类。
  8. 朋友函数和朋友类。
  • 下面是freeCodeCamp关于面向对象编程的视频讲座— 用Python进行面向对象编程

数据库管理系统:SQL查询。

  1. SQL查询。
  2. 规范化—含义，规范化表格的原因，不同的范式。
  3. 无损和有损分解。
  4. 表中不同类型的键—主键、复合键、候选键、超级键。
  5. ER模型-含义和组成部分。
  6. 文件结构— B树，索引。
  7. 并发问题。
  • 下面是Freecodecamp关于SQL的视频讲座——SQL教程初学者全数据库课程

操作系统:操作系统及其类型。

  1. 操作系统及其类型。
  2. 流程管理—流程的属性和状态。
  3. CPU调度算法——FCFS、SJF、SRTF、循环调度、优先级调度。
  4. 过程同步——必要条件、烘焙算法、生产者-消费者问题、用餐哲学家问题、读写问题。
  5. 互斥和信号量——很重要。
  6. 线程—很重要。
  7. 死锁——必要条件、银行家算法、死锁预防、避免、恢复、纠正。
  8. 内存管理——多分区、外部和内部碎片、分页、分段。
  9. 虚拟内存需求分页，页面替换算法，系统颠簸。
  10. 文件分配—连续、链接和索引文件分配。
  11. 磁盘调度算法— FIFO、SCAN、C-SCAN、LOOK、C-LOOK。

第11月和第12月

  • 申请工作。
  • 如果你一直在公共场合分享你的旅程。许多招聘人员会找你。如果他们没有联系你，你需要自己去找他们。
  • 开始在各种门户网站上求职。这是你需要在各种社交平台上申请和接近人的时候。
  • 很多人不会回复你的信息，所以不要失去希望。但是很少有人愿意帮忙。
  • 寻找愿意帮忙的人。请他们推荐你，然后开始准备你的面试。
  • 既然你已经花了很多时间学习数据结构和算法，你需要回去再次温习那些技能。
  • 继续接受采访，回顾你的项目。几个月后，你会得到你的第一份工作，成为一名软件开发人员。
