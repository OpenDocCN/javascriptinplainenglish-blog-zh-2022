## 第一个月

1.  选择任何编程语言。不要浪费时间去过多考虑应该使用哪种编程语言。所有编程语言的基础都是一样的。
2.  如果启动 Python，Java 或者 C++会更好。
3.  Java 学习资源— [Java 8 初学者全教程](https://www.youtube.com/watch?v=grEKMHGYyns)
4.  学习 Python 的资源— [Python 初学者全教程](https://www.youtube.com/watch?v=rfscVS0vtbw&t=3s)
5.  你这个月的目标是关注任何编程语言的基础。你不应该一次专注于一种以上的编程语言。如你所想，你会混淆语法。
6.  不要光看教程就以为自己已经完成了课程。你和导师一起编码。你必须尽自己的努力。您需要进行一些代码更改并查看结果。

## 第二和第三个月

1.  学习数据结构和算法。
2.  可以通过网络课程。你甚至可以在书籍的帮助下做到。
3.  你正在学习这一点，因为你将在各种公司面试中被测试数据结构和算法技能。
4.  一个很好的资源是麻省理工学院的讲座。这是一个很好的资源，可以帮助你理清基础知识。教过这些课程的教授对这门学科了如指掌。
5.  你可以在这里找到课程— [算法和计算](https://www.youtube.com/watch?v=ZA-tUyM_y7s&list=PLUl4u3cNGP63EdVPNLG3ToM6LaEUuStEY)
6.  你不应该一次看完整个课程。你只需要看一场讲座，然后去 Hackerrank 上解决相关的[问题。你也可以选择](https://www.hackerrank.com)[leet code](https://leetcode.com)上的简单问题。
7.  记住，你不可能一下子解决所有的问题。习惯这些问题需要一些时间。
8.  有些情况下，你甚至不能正确理解这个问题。在这些情况下，你不应该失去耐心。有一些问题是和各种话题混搭的。
9.  这两个月将确保你获得解决问题的技能。
10.  确保你不会失去希望。如果你是初学者，这个阶段对你来说会有点困难。

## 第 4 个月

1.  一旦你完成了麻省理工学院讲座中所有的算法和数据结构视频。
2.  你对计算机算法的基础有很深的理解。在这一个月里，如果需要的话，可以快速重温所有的讲座。你必须这么做。
3.  现在剩下的时间应该用来练习你在课程中学到的所有概念的问题。
4.  有成千上万的网站供你练习做题。你只要选一两个，不断练习题就行了。
5.  全球使用的两大网站包括— [Codeforces](https://codeforces.com) 和 [Leetcode](https://leetcode.com)
6.  集中精力解决更多的问题，用你学到的所有概念解决更多的问题。

## 第 5 个月

1.  即使是这一个月，你也要花一半的时间多练习数据结构和算法方面的问题。
2.  你可能想知道为什么我如此关注数据结构和算法。因为这是唯一会在几家公司进行测试的东西。
3.  在任何大型科技公司，这都是成为软件开发人员最难也是最重要的部分。这里还有两个练习问题的网站——[hacker rank](https://www.hackerrank.com)、 [Codechef](https://www.codechef.com)
4.  在这一个月里，你需要弄清楚自己想成为什么样的开发者。成为软件开发人员有很多途径。
5.  你可以选择前端开发，后端开发，iOS 开发，或者如果你想用云。这个月致力于发现你的兴趣，选择你感兴趣的东西。
6.  即使这个月你下不了决心。别担心，我们将在第七个月讨论成为前端、后端和 iOS 开发者的途径。

## 第 6 个月

1.  你应该花这个月的时间学习如何制作网站的基础知识。
2.  对软件开发者来说，了解网络是如何工作的是很重要的。你应该从基本的 HTML、CSS 和 JavaScript 开始。不要跳过这一步，即使你不想成为全职前端工程师。
3.  作为一名软件开发人员，其他开发人员希望您至少了解 HTML、CSS 和 JavaScript 的基础知识。
4.  在 Youtube 上，有成千上万的教程，你可以从中学习这些东西。你可以选择任何资源。当我第一次知道 youtube 上没有多少视频时，我参考了这个标准。
5.  这个月创建基本的网站。如果英语不是你熟悉的语言，你可以选择母语的视频。因为世界各地的导师都可以在 youtube 上找到。

*   学 HTML— [HTML 全教程建网站教程](https://www.youtube.com/watch?v=pQN-pnXPaVg)
*   学习 HTML5 和 CSS3 — [从头开始学习 HTML5 和 CSS3](https://www.youtube.com/watch?v=mU6anWqZJcc)
*   学习 JavaScript — [JavaScript 编程全程](https://www.youtube.com/watch?v=jS4aFq5-91M)
*   这里有一个优秀的 Github 资源库，里面包含一个专业的前端模板，可以创建一个专业的网站。— [HTML5 样板文件](https://github.com/h5bp/html5-boilerplate)

## 第 7 个月

1.  这个月要学的第一件事是关于版本控制。您应该了解版本控制是如何工作的。
2.  Git 和 Github 是前进的方向。先从版本控制的基础开始，再详细了解 GitHub。理解 Github 很重要。
3.  有关 Github 的更多信息，请参见—[Git 和 GitHub for Poets 简介](https://www.youtube.com/watch?v=BCQHnlnPusY&list=PL6E4CnqWlKXP0uiko8-n4d-rndbHZDS_R)。这是一个关于 Git 和 Github 的小老视频。但是老师在帮助你学习概念方面做得很好。

5.  一旦你学会了如何使用版本控制，你就会感到自信。学习 Git 和 Github 的时间不会超过 10 天。
6.  这个月我们有很多空闲时间。记住，我们已经讨论过决定你想成为什么样的开发人员。

## 面向对象编程概念:OOP 概念。

面向对象编程(C++)中涉及的重要主题

1.  哎呀的概念。
2.  多态性的类型。
3.  虚函数——运行时多态性。
4.  继承——类型、虚类、可怕的钻石问题(Imp)。
5.  构造函数和析构函数——虚拟析构函数、私有构造函数和析构函数。
6.  智能指针。
7.  单例类。
8.  朋友函数和朋友类。

*   下面是 freeCodeCamp 关于面向对象编程的视频讲座— [用 Python 进行面向对象编程](https://www.youtube.com/watch?v=Ej_02ICOIgs&t=1s)

## 数据库管理系统:SQL 查询。

1.  SQL 查询。
2.  规范化—含义，规范化表格的原因，不同的范式。
3.  无损和有损分解。
4.  表中不同类型的键—主键、复合键、候选键、超级键。
5.  ER 模型-含义和组成部分。
6.  文件结构— B 树，索引。
7.  并发问题。

*   下面是 Freecodecamp 关于 SQL 的视频讲座——[SQL 教程初学者全数据库课程](https://www.youtube.com/watch?v=HXV3zeQKqGY)

## 操作系统:操作系统及其类型。

1.  操作系统及其类型。
2.  流程管理—流程的属性和状态。
3.  CPU 调度算法——FCFS、SJF、SRTF、循环调度、优先级调度。
4.  过程同步——必要条件、烘焙算法、生产者-消费者问题、用餐哲学家问题、读写问题。
5.  互斥和信号量——很重要。
6.  线程—很重要。
7.  死锁——必要条件、银行家算法、死锁预防、避免、恢复、纠正。
8.  内存管理——多分区、外部和内部碎片、分页、分段。
9.  虚拟内存需求分页，页面替换算法，系统颠簸。
10.  文件分配—连续、链接和索引文件分配。
11.  磁盘调度算法— FIFO、SCAN、C-SCAN、LOOK、C-LOOK。
