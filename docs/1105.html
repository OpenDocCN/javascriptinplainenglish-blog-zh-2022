大局

首先，您需要知道选择器只是一个元素、id或类，用于在HTML文档中选择元素。没什么复杂的。以下是一些例子:

body选择HTML body元素

.className用"className"类(class="className")选择所有元素

#idName选择带有idName id ( id="idName"`)的元素

*选择所有DOM元素

现实生活中的用例并不总是简单的。有些时候，我们需要用许多标准和特征来选择一个特定的事物。这时我们就需要使用CSS组合子了。
